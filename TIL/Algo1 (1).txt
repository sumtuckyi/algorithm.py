1. 이진탐색에 대해 간단히 설명하시오.
이진탐색이란 데이터에서 찾고자 하는 값(이하 key)과 데이터의 중간값의 대소를 비교하여
비교 시마다 탐색할 데이터의 길이를 줄여가는 탐색 방식이다.
따라서 탐색 전에 반드시 데이터가 정렬되어 있어야 한다.
만약 중간값이 key보다 큰 경우에는 중간값을 기준으로 그보다 값이 작은 데이터를 대상으로 다시 중간값을 설정하고 , 중간값이 key보다 큰 경우에는 중간값을 기준으로 그보다 값이 큰 데이터를 대상으로 다시 중간값을 설정하여 탐색을 진행한다.
한 번의 탐색 시마다 탐색할 데이터가 절반으로 줄어들기 때문에 시간 복잡도가 O(logN)으로 효율적인 탐색방식이다. 

2. 이진탐색 코드 작성

def binarySearch(a, N, key):
    start = 0
    end = N -1
    while start <= end:
        mid = (start + end) // 2
        if key == a[mid]:
            return True
        elif key > a[mid]:
            start = mid + 1
        else:
            end = mid - 1
    return False
